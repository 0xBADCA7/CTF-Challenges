This cryptography module implements, and offers some usage for a threshold secret sharing scheme describe by Krawczyk, (more information here http://www.springerlink.com/content/0pdlwl2k4n952e2e ).

This method cleverly combine Shamir's scheme, information dispersal and DES encryption.

This implementation uses the python-poly module for polynomial calculation http://code.google.com/p/python-poly/ and also python-crypto for DES encryption.

The module provides usage for both the "dealer" of the secret and the client of accepting the parts of the secret

Usage Example: 

>>> from secret_sharing import secret_sharing
>>> from secret_sharing_c import secret_sharing_c

>>> S = secret_sharing("The quick brown fox jumps over the lazy dog", 100, 6)

>>> s = secret_sharing_c()
>>> s.add_share(S.share_random())
>>> s.add_share(S.share_random())
>>> s.add_share(S.share_random())
>>> s.add_share(S.share_random())
>>> s.add_share(S.share_random())
>>> s.add_share(S.share_random())

>>> print s.reconstruct()

>>> The quick brown fox jumps over the lazy dog
